# Uzay Hukuku ve Teknolojisi Terimler Sözlüğü

Bu sözlük, projede geçen temel kavramların anlaşılmasını kolaylaştırmak için hazırlanmıştır.

| Terim | Açıklama |
| :--- | :--- |
| **SSA (Space Situational Awareness)** | Uzay Durumsal Farkındalık. Uzaydaki nesnelerin, hava durumunun ve tehlikelerin izlenmesi. |
| **OST (Outer Space Treaty)** | 1967 Dış Uzay Antlaşması. Uzay hukukunun "Magna Carta"sı sayılır. |
| **ITU (International Telecom. Union)** | Uluslararası Telekomünikasyon Birliği. Frekans ve yörünge slotlarını yöneten kurum. |
| **LEO / GEO** | Alçak Dünya Yörüngesi (Low Earth Orbit) / Yer Sabit Yörünge (Geostationary Orbit). |
| **Space Debris** | Uzay Çöpü. İşlevini yitirmiş uydu parçaları ve roket kalıntıları. |
| **Jamming / Spoofing** | Sinyal Karıştırma / Sinyal Sahteciliği (Siber-Elektronik saldırılar). |
| **Common Heritage of Mankind** | İnsanlığın Ortak Mirası İlkesi. Uzay kaynaklarının paylaşımına dair bir doktrin. |
| **STM (Space Traffic Management)** | Uzay Trafik Yönetimi. Uzaydaki hareketlerin koordinasyonu. |
