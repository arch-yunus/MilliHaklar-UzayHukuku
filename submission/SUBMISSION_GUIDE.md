# TUA Astrohackathon Submission Guide

This folder contains everything you need for the "Drive Link" part of your submission.

1.  **presentation_guide.md**: Use this content to create your 7-page slide deck.
2.  **demo_video_script.md**: Use this for a 30s striking video.
3.  **project_banner.png**: High-quality visual for your slides/video.

### Instructions:
-   Download this folder (or link the repo path).
-   If using Google Drive, make the link **PUBLIC (Anyone with the link)**.
-   Copy the text from `SUBMISSION_DRAFT.md` (in root or artifacts) to the "Proje Özeti" and "Proje Detaylı Özet" fields.
-   Provide the GitHub Repo link for the "Proje Dosyası" field.

Good luck! 🛡️🇹🇷🌌
